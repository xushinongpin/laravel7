用户表【可以同时开后台卖产品】
    id 自增id
    username 用户名
    truename  真实姓名 【必须与银行卡一致】
    password   加密后的密码
    remember_token  记住我令牌
    created_at   注册时间
    updated_at   登陆时间
    status 1前台 2后台 3前台停用 4后台停用
    google2fa-谷歌验证 后台必须，并且同时需要绑定邮箱【首次进入要求他绑定，忘记或者遗漏可以邮件找回】
    wechat 微信登陆【需要多少个键待定】
    qq     QQ登陆【需要多少个键待定】
    weibo  微博登陆【需要多少个键待定】
    email  邮箱，pc手机直接注册用于验证与忘记密码谷歌验证等时找回
    phone  手机号 用于验证与忘记密码谷歌验证等时找回
    pid  上级id
    ppid  所有上级id
    bbid   所有下级id
    money  账户余额
    integral  积分
    domain   个人域名
    recommend  推荐商品【 存储商品id（多个都好隔开），如果不存在推荐的商品id则根据默认规则推荐 】

收款类型
    id  自增id
    name  收款类型 【1银行卡 2微信 3支付宝 ......】

收款表
    uid   用户对应银行卡
    cid   收款类型
    bankcard  银行卡或者支付宝微信等账号【 开启后台后需要 】
    cardname   银行卡类型【建设等，银行卡需要，其他待定】
    createtime  添加时间
    updatetime  最后更改时间

关键词关联表【商品标签】
    id  自增id
    name  名称

品牌表
    id  自增id
    name  品牌名称

商品分类表
    id  自增id
    name   分类名称
    keywordsid  关键词对应id组【多个用逗号隔开】
    description   分类描述
    text   分类简介
    pid   上级id
    ppid  上级所有id
    bbid  下级所有id组

商品表【金钱以int类型存储，*100】
    id    自增id
    uid   商家id
    status  商品状态  【1启用 2停用】
    name  商品名称
    keywordsid  关键词对应id【多个用逗号隔开】
    description  商品描述
    sid    分类id
    unit   商品单位
    sales   销量
    bsales  基本销量【作假】
    praise   点击量
    bpraise  基本点击量【作假】
    share   分享量
    bshare  基本分享量【作假】
    stock   库存
    makeaddress  生产地址
    makeaddressmap  生产地址定位
    marketprice   市场价
    sellingprice  销售价
    costprice     成本价
    freight       运费
    limitpurchase  限购【0为不限】
    minpurchase    最少购买量
    integralselect  积分设置【0不能用积分 1只能积分兑换 2积分+现金支付 3 积分或者现金其中一种购买】
    integral   等价积分【1积分=1分钱】
    giftintegral  积分赠送【购买一份赠送多少积分】
    image   商品图片
    video   商品视频 【自己上传或者引入外部视频】
    details   商品详情表

商品规格表
    id   自增id【  编号】
    marketprice   市场价
    sellingprice  销售价
    costprice     成本价
    stock   库存

订单表
    id    编号
    type  类型【普通支付，优惠支付等等】
    paytype  优惠类型  【在线，线下等】
    buyer    买家
    bid     买家id
    delivery  收货信息
    comments   买家留言
    simages  商品图
    sname   商品名称
    sprice   商品价格【单个】
    snumber   商品数量
    samount   实际支付价格【商家可以调整价格，不能低于商品的95%】
    dstatus    【1待发货 2入库中  3发货中  4已发货  5配送中】
    lid   物流id
    loid  物流订单号


物流  logistics
    id  自增id
    name  物流
    price  物流价格【价格规则可以用json存储】


域名绑定表
    domain   域名
    title    标题
    keyword  关键词【多个用逗号隔开】
    description  描述
    uid      对应用户id 【 默认打开全部域名 】
